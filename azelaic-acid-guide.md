# Azelaic Acid: A Versatile Skincare Ingredient

Azelaic acid is a naturally occurring dicarboxylic acid found in grains like barley and wheat. It is highly effective for treating both acne and rosacea.

## Key Benefits

| Feature | Description |
| :--- | :--- |
| **Antimicrobial** | Kills acne-causing bacteria. |
| **Anti-inflammatory** | Reduces redness and swelling. |
| **Comedolytic** | Helps prevent clogged pores. |
| **Skin Brightening** | Inhibits tyrosinase, helping with hyperpigmentation. |
